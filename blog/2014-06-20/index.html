ウェブサイトを開設しました

公開日: 2014年6月20日
最終更新日: 2014年6月21日

数日ほど前からサイトの構築を始めて一応完成したので，GitHub Pages を使って公開することになりました．

この程度のサイトでも，HTML, CSS, JavaScript には不慣れのため一苦労．
Gitも時々思い出したようにしか使わないので，コマンドを忘れてしまっていて結構大変でした．
あとは個人的に，Internet Explorer がHTML5のmain要素に対応していてほしかったです．

さて，愚痴はひとまず置いておいて，このサイトをどのように運用していくかですが，継続的な更新がほぼ必要でなく軽量な記事に関しては“雑記”（要はブログですね）に追加し，追記を行う予定がある，もしくはサイズが大きくなりそうな，独立させることが望ましいと思われる記事はトップページの“コンテンツ”に追加していく方針で，暫くは行ってみたいと思います．

また，私のメールアドレスはフッタ（ページ下部）で公開しているので，御意見・御感想等があれば是非御連絡下さい．
誤字脱字や誤った内容を発見した場合は，GitHubの機能であるIssuesや Pull Request を活用してくださればと思いますが，もちろんメールでも構いませんし，何らかの手段で直接連絡頂けると幸いです．

〜P.S.〜
将来拡張しやすいソースコードを書いたつもりですが，まだ手直しの余地は残されていそうです．
記事が増えHTMLファイルが多くなると修正が困難になるため，暫くはソースコードの修正がメインとなるかもしれません．
